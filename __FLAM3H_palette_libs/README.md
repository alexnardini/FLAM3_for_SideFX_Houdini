### FLAM3H™ Palette presets libraries ###

Those files are the **FLAM3H™** Palette presets libraries.</br>
They are available to load in and familiarize with the Palette tab (**CP** _which stand for Color Palette_).</br>
Each file contain multiple Palette presets, read the HDA documentation to know more about the file format.</br>

Fractal Flames historically use a Palette composed of 256 colors.</br>
**FLAM3H™** allow to go beyound this limit and up to 1024 colors.</br>
Some Palette library files included here are made specifically for this as well for 512 colors.
