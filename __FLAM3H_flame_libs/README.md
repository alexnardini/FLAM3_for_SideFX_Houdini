### FLAM3H™ Flame presets libraries ###

Those files are the **FLAM3H™** Flame presets libraries.</br>
They are available to load in and familiarize with the fractal Flame algorithm and the tool.</br>
Each file contain multiple Flame presets, read the HDA documentation to know more about the file format.

</br>

# Contents

</br>
List of every Flame library files and their Presets contents list.</br></br>

_The number at the end of each preset's name indicates the iteration number that will be used to load the preset_.
